Run the following the steps to see the application in your system

Step 1: Download the files from Git Hub using Git pull

Step 2: Install the all required packages which are mentioned in the Requirement text file

Step 3: Run the "runWebServer.bat" file (if it is a window machine) or Run the "python manage.py  server" command in your Mac terminal 

Step 4: Copy the url which will be displayed in the terminal

Step 5: Paste the url in the browser and play around the application
